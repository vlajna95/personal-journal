# DP PersonalJournal 

A screen-reader accessible, password-protected personal journal for Windows. 


## Features 

- **Three workspaces** – notes, alarms, diaries; each in its own tab 
- **End-to-end encryption** – all content is encrypted with your password; nothing is stored in plain text 
- **Multiple diaries** – create as many named diaries as you like, each with its own independent set of dated entries 
- **Full-text search** – search across notes and diary entries simultaneously; jump straight to the result 
- **Alarms** – set date/time reminders with an optional sound notification 
- **Export** – export notes or any diary to plain text or a formatted document 
- **Customizable keyboard shortcuts** – reassign any menu action to the key combination you prefer 
- **UI translated to different languages** – Catalan, English, Esperanto, Spanish, Basque, French, Italian, Norwegian Bokmål, Brazilian Portuguese, Romanian, Serbian, Swedish 
- **Screen-reader friendly** – built with NVDA, JAWS, and Windows Narrator in mind; every control is labelled and status changes are announced 
- **No admin rights required** – installs to your user profile or runs fully portable from any folder 


## Requirements 

- Windows 10 or Windows 11 (64-bit) 
- No Python nor other additional installations needed – the app is self-contained 


## Installation 

Download the latest `PersonalJournal-*-setup.exe` installation from the **[latest release page](https://github.com/vlajna95/personal-journal/releases/latest)** and run it.  
The installer offers a **standard** install (to `%LOCALAPPDATA%\PersonalJournal`) or a **portable** mode that extracts everything to a folder you choose.
