All shortcuts listed below are the factory defaults. Every action can be reassigned – see [Customising keyboard shortcuts](user-guide#customising-keyboard-shortcuts) in the user guide.

## Navigation

| Action | Shortcut |
| --- | --- |
| Switch to Notes tab | Ctrl+1 |
| Switch to Alarms tab | Ctrl+2 |
| Switch to Diaries tab | Ctrl+3 |

## File

| Action | Shortcut |
| --- | --- |
| Lock journal | Ctrl+Alt+L |
| Search | Ctrl+F or F3 |
| Export | Ctrl+E |
| Backup | Ctrl+Shift+B |
| Customize shortcuts | *(menu only)* |
| Minimize to system tray | Alt+Shift+F4 |
| Exit | Alt+F4 |

## Notes

| Action | Shortcut |
| --- | --- |
| New note | Ctrl+N |

## Alarms

| Action | Shortcut |
| --- | --- |
| New alarm | Ctrl+Shift+N |
| Snooze alarm | Ctrl+Alt+F1 (global shortcut) |
| Stop alarm | Ctrl+Alt+F2 (global shortcut) |

## Diaries

| Action | Shortcut |
| --- | --- |
| New diary | Ctrl+D |
| Manage diaries | Ctrl+Shift+D |
| Go to date | Ctrl+G |

## In dialogs

| Action | Key |
| --- | --- |
| Confirm / OK | Enter |
| Cancel / close | Escape |
| Move between controls | Tab / Shift+Tab |

## Shortcut capture dialog

When reassigning a shortcut the following keys have special behaviour:

| Key | Behaviour |
| --- | --- |
| Escape | Cancel without changing the shortcut |
| Enter | Confirm the displayed combination |
| Tab | Move focus (not captured as a shortcut) |
| Bare modifier key (Ctrl, Alt, Shift alone) | Ignored – a shortcut must include a non-modifier key |
