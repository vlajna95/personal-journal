# DP PersonalJournal

DP PersonalJournal is a screen-reader accessible, password-protected personal journal for Windows. All your notes, diary entries, and alarms are stored in an encrypted database on your own computer – nothing is ever sent to a server.

## Highlights

| Feature | Details |
|---|---|
| **Encryption** | Every piece of content is encrypted with your password before being written to disk |
| **Multiple diaries** | Maintain separate diaries (e.g. Family, Travel, Medical) each with independent dated entries |
| **Notes** | Quick free-form notes with a title, accessible any time |
| **Alarms** | Date and time reminders with optional audio notification |
| **Search** | Full-text search across all notes and all diary entries at once |
| **Export** | Export notes or any diary to a plain-text file |
| **Keyboard shortcuts** | Every action has a shortcut; all shortcuts are reassignable |
| **12 languages** | Catalan, English, Esperanto, Spanish, Basque, French, Italian, Norwegian, Portuguese (BR), Romanian, Serbian, Swedish |
| **Screen-reader support** | Tested with NVDA, JAWS, and Windows Narrator |
| **No admin rights** | Installs to your user profile or runs fully portable |
